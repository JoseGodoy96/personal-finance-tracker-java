# 💰 Smart Expense Tracker (Java)

A smart personal finance tracker built in Java to manage income and expenses, analyze spending patterns, and generate financial insights.

## 🚀 Features

### Core functionality
- Add income and expenses
- Categorize transactions (food, rent, transport, etc.)
- Track balances
- View transaction history

### Analytics
- Monthly financial summary
- Expenses by category
- Highest spending category
- Daily average spending

### Smart features
- Spending prediction based on current usage
- Monthly trend comparison
- Basic financial insights
